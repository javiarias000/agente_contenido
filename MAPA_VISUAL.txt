================================================================================
COMPARATIVA: ANTES vs DESPUÉS
================================================================================

ASPECTO                    ANTES                    DESPUÉS
─────────────────────────────────────────────────────────────────────
Logo positioning           Scale decrease + pad     Safe area + overlay
                           ❌ Puede caer fuera      ✅ Siempre en frame

Transiciones               Concat demuxer (seco)    xfade filter
                           ❌ Corte instantáneo     ✅ Fade 200ms suave

Subtítulos                 Whisper → ffmpeg         Validación robusta
                           ❌ Silent failures       ✅ Logging detallado
                           ❌ Path incorrecto       ✅ Escape correcto

Animación                  Post-processing          Integrada por escena
                           ❌ Estáticos/genéricos   ✅ Keyframes progresivos
                           ❌ Todo el video         ✅ Por escena

Motion metadata            ❌ No existe             ✅ motion.json

Text overlay               ❌ No existe             ✅ drawtext filter

Logo overlay               ❌ No existe             ✅ overlay filter

Error handling             ❌ Silent fallback       ✅ Logging + retry

Performance                Múltiples passes         Single pass
                           ~120 segundos            ~100 segundos
